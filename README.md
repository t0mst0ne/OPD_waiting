Calculate and estimate the waiting time of OPD

門診等待 三長兩短 對醫院, 醫師, 病患都是困擾
「三長」即等掛號、等看診、等批價領藥的時間長，「兩短」則是看診時間短、醫生的話短。

想要解決的問題如下
1. 有沒有辦法根據過去看診資料, 找一個演算法, 來預估看診時間和等待時間 ?
2. 到底是先報到先看比較好, 還是按照掛號順序比較好 ?

Data sample
